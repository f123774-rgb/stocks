🔗 Google Sheets 連接設定

連接 Google Sheets 後，你可以在試算表直接編輯資料，網頁會自動同步。多人協作也更方便。

📋 如何取得 Google Sheets API Key

1
前往 Google Cloud Console：console.cloud.google.com，登入你的 Google 帳號
2
建立新專案（或選擇現有專案），點選左側選單 API 和服務 → 程式庫
3
搜尋 Google Sheets API 並啟用它
4
點選 憑證 → 建立憑證 → API 金鑰，複製金鑰貼到上方欄位
5
開啟你的 Google Sheets，將其設為 任何知道連結的人都可以檢視（共用設定），複製網址中的 ID（/d/ 和 /edit 之間的部分）
6
Sheets 的欄位格式：A=代號, B=名稱, C=族群（用|分隔）, D=描述, E=標籤（用|分隔）
